= New Installations =

Method-A:

1. Go to your wordpress admin "yoursite.com/wp-admin"

2. Login and then access "yoursite.com/wp-admin/plugin-install.php?tab=upload

3. Upload and activate this plugin

4. Now go to admin menu -> appearance -> widgets

5- Drag widget to desired sidebar and save settings

6- Make sure that its working fine for you and don't forget to give your feedback

Method-B:

1.	Download the Authors Posts Widget installation package and extract the files on

	your computer. 
2.	Create a new directory named `Authors Posts Widget` in the `wp-content/plugins`

	directory of your WordPress installation. Use an FTP or SFTP client to

	upload the contents of your Authors Posts Widget archive to the new directory

	that you just created on your web host.
3.	Log in to the WordPress Dashboard and activate the Authors Posts Widget plugin.
4.	Once the plugin is activated, a new **Authors Posts Widget** will appear in your Wordpress admin -> appearance -> widgets.

= Upgrades =

To *upgrade* an existing installation of Authors Posts Widget to the most recent release:

1.	Download the Authors Posts Widget installation package and extract the files on

	your computer. 
2.	Upload the new PHP files to `wp-content/plugins/Authors Posts Widget`,

	overwriting any existing Authors Posts Widget files that are there.

	

3.	Log in to your WordPress administrative interface immediately in order

	to see whether there are any further tasks that you need to perform

	to complete the upgrade.
4.	Enjoy your newer and hotter installation of Authors Posts Widget
